# APP-REACT
'duo shou ji' APP  based on react native

## struction

```
|---- project 
    |---- android                          // android
    |---- ios                              // ios
    |---- node_modules                     
    |---- app                              // js code 
        |---- actions                      // Redux actions
        |---- assets                       // images
        |---- components                   // components for reuse 
        |---- utils                        // global functions or tools
        |---- pages                        // one page one js
        |---- reducers                     // Redux reducers
        |---- store                        // Redux store
        |---- constants                    // constants for reuse
        |---- app.js                       // App entry 
    |---- index.ios.js                     
    |---- index.android.js                 
    |---- package.json                     // package
```
